A hodgepodge of basic survival skills in statistics

Definitions

  • The Sample Space is the set of all possible outcomes.
  • A Random Variable assigns quantities to outcomes . For example a fair dice has one sixth probability for the quantity 6.
  • An Event $$\mathcal{E}&& is a set of outcomes.
  • The Probability Distribution is the cumulative Probability density function and tends to 1 at infinity.
