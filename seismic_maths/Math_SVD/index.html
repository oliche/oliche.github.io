PCA

Preprocessing : each feature has to be centered and scaled.

m = 100; % n samples
n = 21; % nfeatures
k = 5;  % number of desired output features
x = rand(100,21);
Sigma = cov(x);
[U,S,V] = svd(Sigma);
  • Retain only the first $k$ columns of $U$. 
    U_ = U(:,1:k);
  • Apply the transform to the original $x$ vector to get the new features $z$ 
    z = x * U_;
